MacroPrep



In this project we wanted to make something useful that anyone could use to lose weight or prepare for a competation. Making it easier for a user to commite to eating right by having the meal break down done for you.

Authors: Hunter crisp, Scott diffin, Austin Juhl, Narinder Singh

MacroPrep:
- The user can input his macros for the day
- Choose the number of meals he wants to eat
- choose the kinds of food he wants to eat
- Then it will output the meal brake downs with the total intake for the day.

The plateform used: HTML and the rest was done in PHP because we need to pull data from SQL .
